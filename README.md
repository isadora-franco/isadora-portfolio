# Isadora Franco — Portfolio

Portfolio pessoal desenvolvido com HTML, CSS e JavaScript puro.

## Estrutura de Arquivos

```
isadora-portfolio/
├── index.html              → Página inicial (Home)
├── portfolio.html          → Galeria de trabalhos
├── curriculo.html          → Currículo completo
├── contato.html            → Informações de contato
├── trabalho-cheetah1.html  → Página individual: Sistemas Embarcados
├── trabalho-cheetah2.html  → Página individual: Veículo Elétrico
├── trabalho-cheetah3.html  → Página individual: Eletrônica Embarcada
├── css/
│   └── styles.css          → Estilos globais
├── js/
│   └── main.js             → Navegação ativa, hamburguer, animações
└── img/
    ├── isadora-perfil.jpg
    ├── cheetah1.jpeg
    ├── cheetah2.jpeg
    ├── cheetah3.jpeg
    ├── instagram.png
    └── linkedin.png
```

## Funcionalidades

- Navegação com link ativo (destaca a página atual)
- Botão hamburguer para mobile (JavaScript)
- Breadcrumbs em todas as páginas internas
- Animações de entrada (CSS + Intersection Observer)
- Páginas individuais de trabalhos
- Design responsivo (mobile, desktop)
- Hover effects nas miniaturas do portfolio
